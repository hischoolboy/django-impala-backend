
[Unofficial backend package](https://docs.djangoproject.com/en/dev/ref/databases/#using-a-3rd-party-database-backend "Unofficial backend package") for Django.

Usable operations
=================

- Django Queryset.
    - insert record
    - relationship
    - Q object

- Aggregation queries
    - aggregate
    - annotate

- Alter queries
    - alter table
    - alter culumn

- manage commands.
    - test
    - migrate
    - shell(required: impala-shell)
    - sql
    - sqlall
    - syncdb

not supported
=============

- update and delete query.
- index query.
